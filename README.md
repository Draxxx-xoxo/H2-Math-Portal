# H2 Math Portal

## Features
- User Authentication
- Leaderboard
- Random Generate values for each question
- Immediate answer checking

- [Desmos](https://www.desmos.com/api/v1.9/docs/index.html) API (In Development)
- [Vercel](https://vercel.com/draxxxxoxos-projects) (Hosting)
